unsigned  options,
const char *  name 
)

Open and initialize a target.

If a target is already open, close it first.

Parameters:
targetThe type of target to use.
optionsA bitfield of option flags.
nameA string whose purpose is determined by the value of target:
  • logmsg_file: The name of the file to use.
  • logmsg_syslog, logmsg_stdout, logmsg_stderr: A string to prepend to each log message.
Returns:
0 on success or -1 on failure, with errno set appropriately.
