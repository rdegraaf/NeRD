void spc_sanitize_files ( int  maxfd)

Closes all file descriptors above a limit, and ensures that stdin, stdout, and stderr are open.

Parameters:
maxfdThe highest-numbered file descriptor to preserve. Set to 2 or less to close all files except for stdin, stdout, and stderr.
Note:
If stdin, stdout, or stderr is closed and cannot be opened, attempts to log a message through the logmsg facility and calls abort().
