Detailed Description

A wrapper class to provide auto_array with reference semantics.

This allows an auto_array to be assigned or constructed from the result of a function that returns an auto_array by value. This allows constructs such as 

   auto_array<Derived> func_returning_auto_ptr(....);
   ...
   auto_array<Base> ptr = func_returning_auto_ptr(....);
