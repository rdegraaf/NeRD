void drop_priv ( const uid_t  newuid,
const gid_t  newgid 
)

Drops privileges by setting the effective and real UID and GID to specified values.

Parameters:
newuidThe new UID to set, or -1 to leave the UID unchanged.
newgidThe new GUD to set, ot -1 to leave the GID unchanged.
Note:
On failure, logs a message through the logmsg facility and calls abort().
This function was designed to permanently drop privileges from root-owned processes. It may also work for permanently dropping privileges from SETUID-root processes.
This function may not work properly on BSD.
