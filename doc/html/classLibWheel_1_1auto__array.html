Detailed Description

A simple smart pointer for arrays that provides strict ownership semantics in the manner of std::auto_ptr.

Despite what the C++ standard guys will tell you, it isn't always possible to use std::vector in place of an array. For intance, you may need to call a C or external library API that requires an array. This class provides automatic destruction in the manner of std::auto_ptr for true arrays.
