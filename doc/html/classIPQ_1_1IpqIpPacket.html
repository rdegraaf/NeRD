Detailed Description

A complete IPv4 packet received via IpqSocket.

Contains IP header fields and the IP payload.

Note:
This class does not reassemble IP fragments.

std::size_t  buflen 
) [static, protected, inherited]

Create an IpqPacket or one of its subclasses from a libipq packet message.

A packet will be deemed to be an IPv4 packet if it has a full IPv4 header and the data length indicated by the header matches the packet's length. A packet will be deemed to be a TCP or UDP packet if it is an IPv4 packet and it has a full TCP or UDP header.

Returns:
A pointer to a dynamically-allocated IpqPacket, or one of its subclasses. This pointer must be freed using delete, and a verdict should be set on it using IpqSocket::sendResponse().
