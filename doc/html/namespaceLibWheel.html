Read data from a file descriptor.

This function is identical to read(), except that it will resume if interrupted by a signal and will never set EINTR on error. Also, it assumes that fd is in non-blocking mode, and will return -1 and set errno to EAGAIN if no data is available. See the man page for read() for further details.
